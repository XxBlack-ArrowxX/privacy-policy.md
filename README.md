# Política de Privacidad - BotGeist
#0481

**Última actualización:** 17 de abril de 2026

Esta Política de Privacidad describe cómo manejamos la información cuando usas **[Nombre de tu Bot]**.

### 1. Información que recolectamos
- Este bot **no recolecta ni almacena** datos personales de forma permanente.
- Solo lee los mensajes necesarios para ejecutar los comandos que tú uses.
- No guardamos historial de mensajes, IDs de usuario ni ningún dato personal.

### 2. Uso de la información
- La información se usa **solo en el momento** para responder a los comandos.
- Todo se procesa en memoria y se elimina inmediatamente después.

### 3. Compartir información
- No compartimos, vendemos ni entregamos ningún dato a terceros.

### 4. Tus derechos
- Puedes pedir en cualquier momento que dejemos de usar el bot.
- El bot es solo para uso privado entre amigos.

### 5. Contacto
- Cualquier duda: contacta a [tu Discord username] en Discord.

Al usar este bot aceptas esta política de privacidad.  
Podemos actualizarla en cualquier momento sin aviso previo.
